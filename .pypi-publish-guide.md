# PyPI Publish Guide — Ankole Framework

> File ini sengaja hidden (dotfile). Ikuti langkah ini saat kamu siap publish.

## Prerequisites

- Package sudah build sukses (`dist/ankole_framework-2.0.0-py3-none-any.whl`)
- `twine` sudah terinstall (`pip install twine`)
- Akun PyPI sudah terdaftar

## Step 1: Buat Akun PyPI

1. Buka https://pypi.org/account/register/
2. Verifikasi email
3. Aktifkan 2FA (wajib untuk upload)

## Step 2: Buat API Token

1. Login ke https://pypi.org
2. Pergi ke **Account Settings** > **API tokens**
3. Klik **Add API token**
4. Name: `ankole-framework`
5. Scope: **Entire account** (pertama kali) — nanti bisa di-restrict ke project
6. Copy token (dimulai dengan `pypi-...`)

## Step 3: Build Package

```bash
cd E2E_Test_Framework
rm -rf dist/ build/ *.egg-info
python -m build
```

Hasilnya:
- `dist/ankole_framework-2.0.0.tar.gz` (sdist)
- `dist/ankole_framework-2.0.0-py3-none-any.whl` (wheel)

## Step 4: Validasi Package

```bash
python -m twine check dist/*
```

Harus output: `PASSED`

## Step 5 (Opsional): Test Upload ke TestPyPI Dulu

Kalau mau test tanpa publish ke PyPI asli:

```bash
twine upload --repository testpypi dist/* --username __token__ --password pypi-YOUR_TEST_TOKEN
```

- Daftar dulu di https://test.pypi.org
- Buat token terpisah di TestPyPI
- Test install: `pip install -i https://test.pypi.org/simple/ ankole-framework`

## Step 6: Upload ke PyPI (Production)

```bash
twine upload dist/* --username __token__ --password pypi-YOUR_TOKEN
```

Atau simpan token di `~/.pypirc` agar tidak perlu ketik ulang:

```ini
[pypi]
username = __token__
password = pypi-YOUR_TOKEN_HERE
```

Lalu cukup:

```bash
twine upload dist/*
```

## Step 7: Verifikasi

1. Buka https://pypi.org/project/ankole-framework/
2. Test install dari mesin lain:
   ```bash
   pip install ankole-framework
   pip install ankole-framework[web]
   pip install ankole-framework[all]
   ```

## Step 8: Update Version (untuk release berikutnya)

1. Edit `version` di `pyproject.toml`
2. Build ulang: `python -m build`
3. Upload: `twine upload dist/*`

## Tips

- Nama `ankole-framework` sudah di-set di pyproject.toml — jika sudah diambil orang, ganti nama
- Setiap version hanya bisa diupload SEKALI ke PyPI (tidak bisa overwrite)
- Gunakan semantic versioning: `2.0.0` → `2.0.1` (patch) → `2.1.0` (minor) → `3.0.0` (major)
